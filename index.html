Implementasi Paralelisme pada Pipeline Machine Learning

Studi Kasus: Penerapan Paralelisme pada Aplikasi Machine Learning Skala Kecil

Coba Inference Model

Abstrak

Project ini mengevaluasi efektivitas teknik Komputasi Paralel dalam mengatasi latensi tinggi pada pemrosesan dataset gambar masif (20.000 sampel). Tantangan utamanya adalah menyeimbangkan trade-off antara efisiensi waktu dan akurasi model.

Hasil Utama: Implementasi paralelisasi pada CPU 8-Core berhasil memangkas waktu training SVM Hybrid menjadi 58 detik (Speedup 3.42x) dengan akurasi mencapai 86%.

Desain Solusi Paralel

Kami menggunakan pola desain Master-Worker Pattern untuk mendistribusikan beban kerja ke 8 Core CPU secara simultan. Pendekatan ini menggunakan library joblib dengan backend loky (process-based) untuk mengatasi hambatan Global Interpreter Lock (GIL) pada Python.

Evaluasi Kualitas Model

Meskipun metode Tradisional (Pixel-based) lebih cepat dalam preprocessing, akurasinya rendah untuk kebutuhan produksi. Metode Hybrid (ResNet + SVM) dengan bantuan komputasi paralel menjadi solusi optimal.

Kesimpulan

Penerapan teknik Parallel Computing terbukti krusial untuk implementasi Deep Learning pada CPU. Tanpa paralelisasi, proses ekstraksi fitur dan training akan memakan waktu estimasi >25 menit. Dengan paralelisasi, waktu dapat dipangkas menjadi ~13 menit (Speedup keseluruhan ~2.3x), memungkinkan iterasi pengembangan yang lebih cepat tanpa mengorbankan akurasi.
